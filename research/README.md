## How to Use

### 1. Start New Research

1. Copy `template.md`
2. Rename: `[article-title]-notes.md`
3. Move to appropriate subfolder (`insights/`, `resources/`, `pemikiran/`)
4. Fill in article scope, key questions, source strategy
5. Begin research

### 2. Research Process

1. Identify & access sources (government, academic, industry, case studies)
2. Extract key findings using template structure
3. For each source:
   - Document URL & citation
   - Extract key findings & statistics
   - Note relevant quotes
   - Assess relevance to article
4. Synthesize findings:
   - Identify common themes
   - Flag contradictions & nuance
   - Extract original insights (not just summaries)
   - Note gaps & limitations
5. Propose article structure based on research
6. Complete reference list

### 3. Quality Check

Before submitting research to Claude Code, verify:
- [ ] 5-10+ diverse sources
- [ ] All key research questions addressed
- [ ] Statistics sourced & verified
- [ ] Contradictions discussed
- [ ] Original synthesis included
- [ ] Gaps acknowledged
- [ ] Complete reference list
- [ ] Proposed structure logical

### 4. Submit to Claude Code

Once research complete:
1. Save research notes file in this directory
2. Provide file path to Claude Code when requesting article write
3. Claude Code akan read research → synthesize → write article
4. Review draft, provide feedback, iterate

## Template Guide

**template.md** includes:

Use template consistently untuk semua articles. Claude Code akan benefit dari consistent structure.
